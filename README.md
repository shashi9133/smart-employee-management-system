# Smart Employee Management System  
A complete Java console-based application that covers **Core Java, OOP, Collections, JDBC, File Handling, Multithreading, Exception Handling, and Java 8**.  
This project is designed for learning, interview preparation, and practicing real-world Java development concepts.

---

##  Features

###  Employee Management
- Add new employees  
- Update employee details  
- Delete employees  
- Fetch employee details by ID  
- List all employees  
- Filter employees by department  

###  File Handling
- Automatically saves employee records to a text file  
- Maintains a backup of employee data  

###  JDBC (MySQL)
- Stores employees in a MySQL database  
- Uses PreparedStatement, ResultSet, and proper exception handling  
- Small and clean DAO layer  

###  Java Concepts Covered
✔ Object-Oriented Programming  
✔ Encapsulation & Constructor Overloading  
✔ Collections (ArrayList, List, Map)  
✔ Custom Exceptions  
✔ Multithreading  
✔ Stream API & Lambda Expressions  
✔ Layered Architecture (Model → DAO → Service → UI)
